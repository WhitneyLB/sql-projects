# SQL Projects

This repository showcases SQL projects focused on analyzing real-world datasets using relational databases.

The work demonstrates the ability to extract insights, join multiple data sources, and perform data-driven analysis using SQL.

---

## Overview

This project demonstrates SQL skills through structured analysis of multiple datasets, including education, sports, and media data.

Each query set focuses on solving specific data questions using joins, aggregations, filtering, and subqueries.

---

## Skills Demonstrated

- Data querying and filtering (SELECT, WHERE, AND, OR)
- Data aggregation and analysis (GROUP BY, HAVING, AVG, COUNT)
- Relational data modeling and joins (INNER JOIN, LEFT JOIN)
- Subqueries and nested logic
- Sorting and ranking data (ORDER BY)
- Pattern matching (LIKE)

---

## Tools Used

- SQLite

---

## Project Structure

- **querying/** → Foundational SQL queries including SELECT, WHERE, ORDER BY, and aggregations  
- **relating/** → Multi-table queries using joins and relational analysis  

---

## Featured Projects

### 📊 DESE School Data Analysis
- Analyzed Massachusetts school data across districts, schools, expenditures, and staff evaluations  
- Used multi-table joins to connect datasets  
- Identified high-performing districts based on spending and staff performance  
- Applied subqueries to compare values against overall averages  

### ⚾ Baseball Players Analysis
- Queried player statistics to identify trends and performance insights  
- Used filtering, sorting, and aggregation to analyze player data  

### 📺 TV Shows Analysis
- Analyzed episode-level data using filtering and sorting techniques  
- Extracted insights from structured television dataset
